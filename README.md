# SOBRE O PROJETO

Esté é um desafio para praticar o uso de API externas através do fetch do curso da EBAC - Front-End. Os arquivos deste repositório são para a entrega do exercício do Módulo 8.

## OBJETIVO:
- Desenvolver uma aplicação web que permita cadastrar, listar e excluir clientes usando a API do CrudCrud. O objetivo é praticar o consumo de APIs com Fetch API, manipulando requisições GET, POST e DELETE.

## Arquivos obrigatórios:
- Arquivo INDEX.HTML (Interface da aplicação);
- Arquivo SCRIPT.JS (Código JavaScript para interagir com a API).

## Funcionalidades exigidas:
- Cadastrar Cliente: O usuário pode inserir um nome e um e-mail e salvar na API.;
- Listar Clientes: Todos os clientes cadastrados são exibidos na tela;
- Excluir Cliente: Criar um botão para remover um cliente da API.

### Adicionais:
- Arquivo STYLE.CSS (estilização da página).
- Ao enviar os dados para API, uma função extra de limpar os campos foi adicionada;
- Botão de limpar os campos, sem a necessidade de enviar os dados para a API;
- Função para exibir uma mensagem de lista vazia quando não houver nenhum dado para listar;
- Função auxiliar para adicionar cliente, tanto na lista, quanto para envio para a API.
